Ndriç është një hapsinë virtuale kushtuar matematikës.

Përmes Ndriç synojmë që të gjithë nxënësit, pa marrë parasysh cilës etapë të nxënies gjinden, të kenë çasje në spjegime të denjë.

Ndriç synon që të gjitha burimet matematikore që mund të nevojitën të jenë të çasshme në gjuhën shqipe pa marrë parasysh vendndodhjën apo mundësitë e nxenësit.

Mirësevini në rrugëtimin tuaj matematikorë!
